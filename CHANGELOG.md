## [0.1.0] - 2026-01-27

### Added
- Initial release of brdoc library
- CPF validation, formatting, and generation
- CNPJ validation, formatting, and generation
- Support for various input formats (formatted, unformatted, with spaces)
- Hashable CPF and CNPJ objects for use in sets and dictionaries
- Comprehensive test suite with 90%+ coverage
- Full type hints support
- Detailed documentation and examples

### Features
- Pure Python implementation with no dependencies
- Fast and reliable validation algorithms
- User-friendly API with both class methods and instance methods
- Clear error handling with custom exceptions
- Formatted output with proper Brazilian document formatting
